프로그래밍 기반 '자동화 데이터 파이프라인' 구축

"해당 데이터셋은 수동 작업이 아닌, Python 기반으로 자체 설계한 make_massive_dataset.py 자동화 스크립트를 통해 생성되었습니다. 본 데이터 생성 파이프라인은 3단계의 핵심 알고리즘으로 구성되어 있습니다."

  • Step 1. 기본 조합 및 구조화 (Combinatorics)
    난수 생성 알고리즘을 활용하여 드론의 6자유도 기동(전진, 후진, 좌우 회전, 상하강) 파라미터와 1~100 범위의 거리(m) 및 각도(도) 수치를 무작위로 교차 결합하였습니다.
  • 이를 통해 언어 모델 학습의 기반이 되는 프롬프트(Instruction)와 기계어 출력(Output) 매핑 데이터의 구조적 형태를 확립했습니다.

Step 2. 음성 노이즈 주입 단계

  • 오인식 매핑 테이블 구축
    야외 환경의 모터 소음 및 풍절음으로 인한 STT 오인식 패턴을 정량적으로 분석하여, Dictionary 자료 구조 형태의 노이즈 사전을 구축했습니다. (예: "착륙" ➔ "장육")
  • 확률적 노이즈 주입 알고리즘 적용
    생성된 텍스트 데이터를 순회하며 70%의 난수 확률로 특정 단어를 사전 정의된 오인식 텍스트로 치환(Replace)합니다.
  • 통제된 환경의 정제된 텍스트를 넘어, 실제 운용 환경의 음향적 특성을 모사하여 강건한(Robust) 훈련 데이터를 프로그래밍 방식으로 증강합니다.
# 파이썬 스크립트 작동 원리 (내부 구조 요약) import random import json # 1. 오인식 매핑 테이블 (노이즈 사전) noise_dict = { "착륙해": ["장육해", "창뉵해", "착육해"], "전진": ["전징", "점진", "앞으로가"] } # 2. 대규모 데이터셋 자동 생성 루프 with open("massive_drone_knowledge.jsonl", "w") as f: for i in range(150000): # 기본 기동 명령 생성 distance = random.randint(1, 100) command_text = f"전진 {distance}m" output_code = f"GO_FORWARD_{distance}" # 3. 70% 임계값 기반 확률적 노이즈 주입 if random.random() < 0.70: # 원문 텍스트를 오인식 매핑 데이터로 치환 noise = random.choice(noise_dict["전진"]) command_text = command_text.replace("전진", noise) # 4. JSONL 포맷 직렬화 및 파일 출력 data_line = {"instruction": command_text, "output": output_code} f.write(json.dumps(data_line) + "\n")

Step 3. 복합도 조절 및 직렬화

  • 학습 복합도 분배 최적화
    단방향의 '단일 기동 명령'과 방향 전환을 수반하는 '복합 기동 명령'을 5:5의 비율로 교차 구성하여 모델의 파라미터 튜닝 효율 및 학습 난이도를 최적화하였습니다.
  • 표준 규격 직렬화 (Serialization)
    생성된 최종 데이터는 대규모 언어 모델(LLM) 파인튜닝 표준 포맷인 JSONL(JSON Lines) 형태로 직렬화하여 영구 저장소에 구축 완료하였습니다.
